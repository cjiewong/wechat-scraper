# 使用指南

## 1. 启动

```bash
cd wechat-scraper
npm install
npm start
```

## 2. 配置公众号平台会话

1. 点击应用顶部「打开公众号平台」
2. 在浏览器登录 `https://mp.weixin.qq.com/`
3. 点击「扫码登录」，在弹窗中用微信扫码并确认
4. 登录成功后会自动回填 `token` 与 `Cookie`
5. 如需手动方式，仍可自行粘贴会话并点击「保存会话」
6. 点击「检测会话」

> 说明：当前版本使用公众号平台官方接口，不再依赖搜狗微信页面抓取。

## 3. 搜索公众号与加载文章

1. 在「公众号与文章」输入公众号名称并点击「搜索」
2. 选择一个公众号
3. 设置时间范围、页数、每页数量
4. 点击「加载文章列表」

## 4. 导出文章

1. 勾选需要导出的文章（默认全部勾选）
2. 选择输出目录
3. 选择导出格式（Markdown / PDF）
4. 选择图片处理方式（保留原图链接 / 上传 COS）
5. 点击「开始导出」

导出期间可在右侧查看进度、统计和运行日志。

## 5. 常见问题

### 会话检测失败

- `token` 或 `Cookie` 已过期，请重新登录公众号平台并更新
- `Cookie` 缺少关键字段（例如 `slave_sid`、`token`）

### 搜索公众号失败

- 会话无效或接口频率受限
- 稍后重试，或降低请求频率

### 导出时图片未上传

- COS 未配置，系统会自动回退为保留原图链接
- 检查 `WECHAT_SCRAPER_COS_CONFIG` 或默认配置文件路径

### PDF 生成失败

- 建议先尝试 Markdown 导出定位问题
- 检查输出目录写权限和磁盘空间
