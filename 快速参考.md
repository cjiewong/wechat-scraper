# 快速参考

## 启动

```bash
npm start
```

开发模式：

```bash
npm run dev
```

## 最短流程

1. 打开公众号平台并登录
2. 点击「扫码登录」自动获取 `token + Cookie`
3. 检测会话
4. 搜索公众号并加载文章
5. 选择输出目录和格式
6. 开始导出

## 关键配置

`modules/config.js`

- `scraper.delayBetweenArticles`: 文章抓取间隔
- `scraper.timeout`: 请求超时
- `scraper.defaultMaxPages`: 默认抓取页数

## 相关文件

- 主进程：`main.js`
- 抓取逻辑：`modules/scraper.js`
- UI：`renderer/index.html` + `renderer/renderer.js`
- 日志：`logs/`

## 当前接口

- 搜索公众号：`/cgi-bin/searchbiz`
- 文章列表：`/cgi-bin/appmsgpublish`
- 内容抓取：文章原文 URL（`mp.weixin.qq.com/s/...`）
